# Projeto de Análise Exploratória de Dados com Streamlit

Este projeto foi desenvolvido para realizar uma análise exploratória de dados a partir de um conjunto de dados de veículos. A aplicação foi criada com Python, utilizando Pandas para manipulação dos dados, Plotly Express para visualizações gráficas e Streamlit para a construção da interface web interativa.

O principal objetivo deste projeto é permitir a visualização simples e rápida de informações do conjunto de dados, ajudando na compreensão de padrões, distribuições e possíveis relações entre variáveis. A aplicação inclui recursos como a geração de histograma e gráfico de dispersão, facilitando a exploração visual dos dados pelo usuário.

Além disso, o projeto também serve como prática de desenvolvimento de aplicações web com Python, integração com GitHub e publicação online.
